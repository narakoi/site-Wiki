终端执行：

```
sudo gedit /etc/default/grub
```

其中，`GRUB_DEFAULT` 这一项表示默认的启动项，后面的值从零开始编号，分别是你在 Grub 引导界面看到的启动项。更改为你需要的启动项序号即可。`GRUB_TIMEOUT` 这一项是超时时间，单位为秒。

更改完毕后：

```
sudo update-grub
```
