终端：

```
sudo  blkid
```

显示出分区信息，记下要自动挂载的分区 UUID，与 Type。

在 Ubuntu 文件系统中创建文件夹作为分区的挂载点。例如：

```
mkdir ~/HDD
```

然后

```
sudo gedit /etc/fstab
```

在最后添加一行

```
UUID=<UUID>	/path/to/HDD	<Type>	defaults	0	0
```

替换其中的 `<UUID>` 与 `<Type>`  为之前的查询结果，重启电脑，则分区已经挂载在 ~/HDD 目录下。
