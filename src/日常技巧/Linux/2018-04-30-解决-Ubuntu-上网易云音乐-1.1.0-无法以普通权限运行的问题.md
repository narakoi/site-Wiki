安装网易云音乐 1.1.0 版本后使用快捷方式可能无法正常启动，但是在终端使用 `sudo netease-cloud-music` 却可以启动，这是权限的问题。

解决方法：

```
sudo chmod 777 -R ~/.cache/netease-cloud-music
```

重启即可。

不懂网易云这么设计的原因是什么，有病。
