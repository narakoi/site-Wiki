用宏代码解决。

视图 - 宏 - 查看宏，输入宏名，点击创建。在 `End Sub` 之前插入

```
For Each templ In ActiveDocument.ListTemplates

For Each lev In templ.ListLevels

lev.Font.Reset

Next lev

Next templ
```

保存运行该宏即可。
